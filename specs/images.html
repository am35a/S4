Images

Responsive

To make an image responsive use class .img-fluid.

Thumbnails

The .img-thumbnail class wrap an image with a background. Background colors of thumbnail can change by .bg-# color classes.

Figures

Use tag figure to add information to an image. Class .figure use with tag figure, class .figure-img use with tag img and for figcaption S4 has the .figure-caption class.
