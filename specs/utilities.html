Background

Background color classes list in S4: .bg-primary, .bg-secondary, .bg-success, .bg-info, .bg-warning, .bg-danger, .bg-light and .bg-dark.

For buttons and links background color move to shadow when mouse :hover or link in :focus.

Extra colorize classes

.bg-blue, .bg-indigo, .bg-purple, .bg-pink, .bg-red, .bg-orange, .bg-yellow, .bg-green, .bg-teal, .bg-cyan, .bg-white, .bg-gray and .bg-black.

Clearfix

Use .clearfix class to expand container around any float elements.

Display

Display property can change by .display-none, .display-unset, .display-inline, .display-inline-block, .display-block, .display-flex, .display-inline-flex, .display-grid, .display-inline-grid, .display-table, .display-table-row, .display-table-cell. Also all properties are responsive with classes .display-sm-#, .display-md-#, .display-lg-# and .display-xl-#

Overflow

For configuring how content overflows an element S4 has .overflow-auto, .overflow-scroll, .overflow-hidden classes also .overflow-#-x and .overflow-#-y has too.

Position

Positioning classes are available in S4 as .position-static, .position-relative, .position-absolute, .position-fixed and .position-sticky. Also S4 manualy can move objects into relative position (left, top, right and bottom) frame by classes .l-0, .t-0, .r-0 and .b-0.

Spacing

S4 include utility classes .m#-#-# to modify margin and .p#-#-# to modify padding.

Margin

Use this patterns for margin classes:

    The first hash can one of:
  • t - for classes margin-top
  • b - for classes margin-bottom
  • l - for classes margin-left
  • r - for classes margin-right
  • x - for classes both margin-left and margin-right
  • y - for classes both margin-top and margin-bottom
    The second hash can one of:
  • sm - for breakpoint 576px
  • md - for breakpoint 768px
  • lg - for breakpoint 992px
  • xl - for breakpoint 1200px
    The third hash can one of:
  • 0 - use for killing margin
  • 1 - for classes .25rem
  • 2 - for classes .5rem
  • 3 - for classes 1rem
  • 4 - for classes 1.5rem
  • 5 - for classes 3rem
  • auto - for classes that set the margin to auto
